This method uses config.ini file under the same directory.
"Resultfile" is the result of other methods. Each row is composed of "start", "end", "start end", "gain" or "loss", of which the third column is only used for placeholder,
can be set any value, and these four columns are directly separated by tabs.
"Bamfile" is the path of the BAM file, "chrfile" is the path of the corresponding reference fasta file, "binlen" is the bin length, and the default value is 1000.
